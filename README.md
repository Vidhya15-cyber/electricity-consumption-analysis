# ⚡ Electricity Consumption Analysis

## 📌 Overview

Electricity Consumption Analysis is a Python-based data analysis project that explores electricity usage data to identify consumption patterns, peak usage periods, and energy trends. The project applies Exploratory Data Analysis (EDA) and visualization techniques to generate actionable insights for energy management.

---

## ✨ Features

- ⚡ Electricity consumption analysis
- 📊 Exploratory Data Analysis (EDA)
- 📈 Energy usage trend visualization
- 🕒 Peak consumption analysis
- 📉 Statistical summaries
- 📋 Energy consumption insights

---
